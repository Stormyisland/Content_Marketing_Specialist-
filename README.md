# Content Marketing Specialist AI Persona

This repository contains a JSON-based AI persona for a **Content Marketing Specialist** designed to assist with content strategy, copywriting, SEO, and campaign planning.

## Overview

The persona, **Alex Mercer**, represents a senior content marketer with deep expertise in B2B SaaS, AI trends, and data-driven storytelling. The JSON file includes tone, voice, tools, goals, and sample outputs to guide AI interactions.

## Use Cases

- **Content Planning**: Generate blog topics, editorial calendars, and distribution strategies.
- **Copywriting**: Draft email sequences, landing pages, and social posts aligned with brand voice.
- **SEO Optimization**: Get keyword recommendations and on-page SEO tips.
- **Performance Analysis**: Interpret analytics and suggest improvements.
- **Lead Generation**: Create lead magnets and CTAs that convert.

## How to Use

1. Load the JSON file into your AI system (e.g., OpenAI GPT, Claude, or custom chatbot).
2. Use the `persona` fields to set system instructions or prompt context.
3. Reference the `sample_output` for style and tone examples.
4. Customize the `goals` and `knowledge_areas` to match your specific industry or campaign.

## Example Prompt

> "You are Alex Mercer, a Senior Content Marketing Specialist. Write a 500-word blog post on how AI is changing content personalization. Include a compelling intro, 3 key points, and a CTA to download a checklist."

## Customization

Feel free to edit the JSON file to:
- Change the persona's name, company, or tone
- Add or remove tools and expertise
- Update goals based on your current KPIs
- Modify response style preferences
